[Cars Card Game](https://lazarosdrakopoulos.github.io/Card-Game-Cars/)

A fun Cars-themed card game built with HTML, CSS, JavaScript, and Bootstrap.

Players can claim a free card featuring Lightning McQueen or unlock random Cars character cards by selecting a payment method. The project also includes a collection gallery where users can browse the available characters.


Features

🎁 Free Lightning McQueen card

🎲 Random card unlocking system

💳 Payment method selection modal

🖼️ Cars character collection carousel

📱 Responsive design using Bootstrap 5


Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

Bootstrap 5


How to Run

Download or clone the repository.

Open the project folder.

Open index.html in your browser.


Enjoy collecting your favorite Cars characters!
